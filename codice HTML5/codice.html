HTML5

L'HTML5 è un linguaggio di markup per la strutturazione delle pagine web, pubblicato come W3C Recommendation da ottobre 2014.

Storia

Lo sviluppo venne avviato dal gruppo di lavoro Web Hypertext Application Technology Working Group. Il progetto è di sviluppare le applicazioni web, con il miglioramento del codice HTML.

Nel 2007 il WHATWG(gruppo di lavoro Web Hypertext Application Technology Working Group) ha collaborato con il W3C ( World Wide Web Consortium) per poi dividersi nel 2012 e creare due versioni di HTML5.

  • W3C il 17 dicembre 2012=prima Candidate Recommendation.
  • 28 ottobre 2014= prima versione dello standard.

    • Novità

      Le novità introdotte dall'HTML5 rispetto all'HTML4 sono:

      • migliorare il disaccoppiamento fra struttura
      • definite dalle direttive di stile
      • supporto per la memorizzazione locale di grandi quantità di dati scaricati dal web browser
